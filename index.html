Afterburner

A proof-of-concept in-browser JavaScript-based Analytical RDBMS

Afterburner is a proof-of-concept in-browser relational analytics system that explores the idea of implementing an analytical RDBMS in pure JavaScript so that it runs completely inside a browser with no external dependencies. Afterburner, generates compiled query plans that exploit typed arrays and asm.js, two relatively recent advances in JavaScript.

For more details, see:

  • Kareem El Gebaly and Jimmy Lin. Afterburner: The Case for In-Browser Analytics arXiv:1605.04035, May 2016.

In this demo we have prepared for you two files for the TPC-H tables at 1GB scale.

  • lineitem table
  • orders table

We have tested Afterburner using the following web browsers:

  • FireFox (64-bit)
  • Google Chrome (64-bit)
  • Opera (64-bit)

We recomend using clients with 6-8 GB of physical memory.
